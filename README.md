### Bluebell CSL - Wavefront .obj

This repo stores the entirety of the Bluebell CSL library, a set of non-flyable aircraft models for multiplayer X-Plane plugins, converted to Wavefront .obj objects. This is a universal format, allowing the use of the aircraft for many different purposes. This library was converted using the following tool: https://github.com/isaacbarker/obj8-to-wavefront-obj. I have released this with the same license as listed by the original project. See releases for the files.

An example of the results can be found at: https://github.com/isaacbarker/skydome

_The original models can be found here: https://forums.x-plane.org/files/file/37041-bluebell-obj8-csl-packages/, and at the following git repo: https://github.com/oktal3700/bluebell_
